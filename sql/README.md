# Guía SQL del Proyecto

## Archivos incluidos
- `create_database.sql`: crea la estructura principal de la base de datos.
- `datos.sql`: carga datos de ejemplo para pruebas funcionales.

## Orden recomendado de ejecución
1. Crear la base de datos PostgreSQL.
2. Ejecutar `create_database.sql`.
3. Ejecutar `datos.sql`.

## Tablas principales
- usuarios
- productos
- carrito
- carrito_productos
- pedidos
- detalle_pedidos
- metodos_pago
- transacciones_pago

## Nota
Este directorio concentra los scripts necesarios para levantar el entorno de datos del proyecto.
