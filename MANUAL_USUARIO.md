# Manual de Usuario

## 1. Objetivo
Este manual explica las funciones principales del sistema y se enfoca en el proceso de compra, especialmente en la pasarela de pagos.

## 2. Ingreso al sistema
1. Abre la aplicación en el navegador.
2. Ingresa con tu correo electrónico y contraseña.
3. Si no tienes cuenta, selecciona **Regístrate aquí** para crear una nueva.

## 3. Funciones principales
- Consultar y filtrar productos.
- Agregar productos al carrito.
- Modificar cantidades o eliminar productos del carrito.
- Finalizar la compra mediante la pasarela de pagos.
- Revisar tus pedidos y transacciones.

## 4. Pasarela de pagos
La pasarela de pagos es el último paso antes de confirmar la compra.

### 4.1. Cómo acceder
1. Ingresa al carrito.
2. Revisa el resumen.
3. Selecciona **Ir a pagar**.

### 4.2. Selección del método de pago
El sistema muestra los métodos disponibles según el módulo de pagos. Debes elegir uno antes de continuar.

Los métodos pueden incluir:
- Tarjeta.
- Billetera digital.
- Transferencia simulada.

### 4.3. Datos requeridos
Dependiendo del método seleccionado, el sistema solicitará datos diferentes:
- Tarjeta: titular, número, vencimiento y CVV.
- Billetera digital: teléfono y código de verificación.
- Transferencia: datos de confirmación según la configuración del sistema.

### 4.4. Validaciones
El sistema verifica automáticamente:
- Que los campos obligatorios estén completos.
- Que el formato de los datos sea correcto.
- Que exista stock suficiente.
- Que el monto total coincida con el carrito.

### 4.5. Resultado del pago
Después de enviar la operación, puedes obtener dos resultados:
- **Pago aprobado**: el pedido se genera y el stock se descuenta.
- **Pago rechazado**: la transacción no se completa y puedes volver a intentarlo.

### 4.6. Qué hacer después del pago
- Si el pago fue aprobado, puedes revisar el pedido en **Mis pedidos**.
- También puedes revisar el historial de la operación en **Mis pagos** o **Mis transacciones**, según tu perfil.

## 5. Consulta de pedidos y transacciones
- **Mis pedidos**: muestra el detalle de la compra realizada.
- **Mis pagos / Mis transacciones**: muestra el estado del pago, método usado y código de transacción.

## 6. Recomendaciones de uso
- Verifica la cantidad antes de pagar.
- No intentes superar el stock permitido por producto.
- Utiliza datos válidos en la pasarela para evitar rechazos.
- Revisa los detalles del pedido apenas se complete la operación.
